Build a mobile app that exercises the woodpecker training method with Chess. The user will be presented with a set of puzzles to solve.

Use the Lichess Puzzle API https://lichess.org/api#tag/Puzzles/operation/apiPuzzleId  to retrieve the puzzle contents given the provided default-collection.json . The URL will be in this format https://lichess.org/api/puzzle/{id} . When you make a GET request it will return a response with the structure indicated on their API site. When you retrieve the Lichess Puzzle, be kind to Lichess, and cache the response. The puzzle solution and initial position will always remain the same. Store the minimal amount needed for the user to interact with the puzzle. If we have the Lichess Puzzle in cache, do not make the GET call to Lichess API. Also do not send all 200 requests at the same time.

Make a clean simple UI with some dark mode theming. Blacks, reds, blues, oranges, etc. The user will be given an interactable chessboard and attempt each puzzle's solution. Check the Lichess Puzzle API for the solution under puzzle.solution returned. When the user first lands on the page, simply give them a button to manage their sessions. If its their first time, they'll only see Start Session. Otherwise there can be Resume Session, Pause / Stop Session. Keep track of how many puzzles were attempted, completed successfully, and failed for each session. Also keep track of total time elapsed. For any puzzles failed, provide the URL to Lichess where they can try again. When the user clicks Stop Session they should see a summary report of their session - Total Time, Puzzles successful (by category), Puzzles failed (by category), and list of puzzles they failed with the links to Lichess actual puzzle URL (not the API one) which has this structure https://lichess.org/training/<ID>.

Note that Lichess Puzzle API returns the initial move state as PGN. This is used to setup the initial board state. And the solution is in UCI format. You can convert it to SAN format using chess.js. An example here:

// Example UCI move
var uciMove = "e2e4";
// Parse the UCI move
board.push(uciMove);
// Convert to SAN
var sanMove = board.moveHistory[board.moveHistory.length - 1].san;
console.log(sanMove); // Outputs "e4"
